API

Convenience tools

myimagelib.readdata(folder[, ext, mode])

Read all files in a folder with certain extension.

myimagelib.bestcolor(n)

Default plot color scheme of Matplotlib and Matlab.

myimagelib.show_progress(progress[, label, ...])

Display a progress bar in command line environment, which looks like

myimagelib.xy_bin(xo, yo[, n, mode, bins])

Bin x, y data on log or linear scale.

Image processing

myimagelib.to8bit(img)

Auto adjust the contrast of an image and convert it to 8-bit.

myimagelib.rawImage(file_dir)

Convert raw images (e.g. nd2) to tif sequences.

myimagelib.imfindcircles(img, radius[, ...])

Find circles in images.

PIV

myimagelib.compact_PIV(data)

Compact PIV data structure.
