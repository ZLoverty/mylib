myimagelib.myImageLib.bpass¶

myimagelib.myImageLib.bpass(*args)¶

Apply bandpass filter on images. Useful when raw images have long wavelength intensity gradient.

Parameters

  • img (2d array) – 8-bit image

  • low (int) – lower limit wavelength

  • high (int) – upper limit wavelength

Returns

processed image with low and high wavelength signals filtered

Return type

2d array
