myImageLib.xy_bin

myImageLib.xy_bin#

myImageLib.xy_bin(xo, yo, n=100, mode='log', bins=None)#

Bin x, y data on log or linear scale

Parameters

  • xo – input x

  • yo – input y

  • n – points after binning

  • mode – “lin” or “log”, scale to bin the data

  • bins – set the bins to bin data together

Returns

  • x: binned x

  • y: means in bins
