myimagelib.corrLib.plot_gnf¶

myimagelib.corrLib.plot_gnf(gnf_data)¶

Used for small scale test of gnf analysis. It incorporates the guide of the eye slope already in the function, which usually needs further adjustment when preparing paper figures.

Parameters

gnf_data – gnf data generated by df2()

Return ax

axis on which the data are plotted

Return slope

the slope of the gnf_data curve
