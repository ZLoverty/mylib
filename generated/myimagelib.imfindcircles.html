myimagelib.imfindcircles

myimagelib.imfindcircles(img, radius, smooth_window=11, sensitivity=0.85)

Find circles in images. The algorithm is based on Hough Transform and the idea of Atherton and Kerbyson 1999, using the edge orientation information to improve the performance on noisy images.

Parameters:

  • img (numpy array) – Input image. Note that it has to be grayscale and 8-bit, otherwise the function will raise an error.

  • radius (list) – Radius of circles to detect. A list of 2 values, [minimum, maximum].

  • sensitivity (float) – Sensitivity for circle detection. The sensitivity takes values in (0, 1], and higher value leads to more detected circle.

Returns:

DataFrame with columns [x, y, r] for the centers and radii of detected circles.

Return type:

pandas.DataFrame
